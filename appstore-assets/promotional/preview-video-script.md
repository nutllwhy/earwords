# App Store 预览视频脚本

## 视频规格

| 规格 | 要求 |
|------|------|
| **时长** | 25-30秒 |
| **分辨率** | 与截图对应 (6.7"/6.5"/5.5") |
| **帧率** | 30fps |
| **格式** | MOV / MP4 |
| **音频** | 背景音乐 + 可选音效 |

---

## 通用视频脚本 (30秒)

### 镜头1: 品牌开场 (0-3秒)
**时长**: 3秒
**画面**: 
- App图标从中心放大
- 品牌名称 "EarWords" 淡入
- 副标题 "雅思词汇学习" 淡入
**动效**: 缩放 + 淡入
**音效**: 简洁的品牌音效

### 镜头2: 学习界面展示 (3-8秒)
**时长**: 5秒
**画面**:
- 手机界面显示学习卡片
- 单词 "abandon" 正面展示
- 卡片翻转展示释义和例句
- 用户点击评分按钮
**动效**: 卡片翻转动画 + 按钮点击反馈
**音效**: 卡片翻转音效 + 轻柔点击音
**文字**: "科学记忆算法 · SM-2间隔重复"

### 镜头3: 磨耳朵功能 (8-15秒)
**时长**: 7秒
**画面**:
- 切换到磨耳朵播放器界面
- 播放按钮点击，进度条移动
- 声波可视化效果
- 切换到后台播放示意
**动效**: 播放动画 + 声波律动
**音效**: 背景音乐渐强
**文字**: "磨耳朵训练 · 随时随地听音学词"

### 镜头4: 学习统计 (15-22秒)
**时长**: 7秒
**画面**:
- 统计界面展示
- 今日数据卡片
- 连续打卡动画 (数字增长)
- 学习趋势图表绘制
**动效**: 数字滚动 + 图表绘制动画
**音效**: 数据提示音效
**文字**: "数据统计 · 可视化学习进度"

### 镜头5: 词库展示 (22-27秒)
**时长**: 5秒
**画面**:
- 章节列表界面
- 快速滚动展示22个章节
- 搜索功能演示
**动效**: 列表滚动 + 搜索展开
**音效**: 滑动音效
**文字**: "3674词汇 · 22章节系统覆盖"

### 镜头6: 行动号召 (27-30秒)
**时长**: 3秒
**画面**:
- App Store下载按钮
- 品牌Logo + Slogan
- "立即下载" 文字
**动效**: 按钮脉动 + 淡入
**音效**: 品牌音效结尾
**文字**: "EarWords 听词 · 雅思备考从听开始"

## 不同尺寸适配

### iPhone 6.7" / 6.5" (纵向)
- 标准纵向布局
- 手机mockup居中显示
- 文字在顶部或底部

### iPhone 5.5" (纵向)
- 相同比例，可共用素材
- 调整文字大小

### iPhone (横向 - 可选)
- App横屏模式展示
- 左右分屏布局

---

## 制作工具建议

1. **Screen Recording**: 
   - iOS Simulator 录屏
   - QuickTime Player (Mac)
   - Reflector / AirServer

2. **视频编辑**:
   - Final Cut Pro (专业)
   - Adobe Premiere Pro
   - DaVinci Resolve (免费)
   - iMovie (简单)

3. **动效设计**:
   - After Effects
   - Motion
   - Figma (简单动画)

4. **手机Mockup**:
   - Rotato
   - CleanShot X
   - Angle (Figma插件)

---

## 背景音乐建议

- **风格**: 轻快、积极、科技感
- **节奏**: 120-130 BPM
- **情绪**: 激励、专注、高效
- **版权**: 使用免版权音乐 (Epidemic Sound, Artlist, YouTube Audio Library)
