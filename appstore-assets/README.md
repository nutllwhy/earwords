## 🎬 应用预览视频

| 规格 | 要求 |
|------|------|
| **时长** | 15-30秒 |
| **格式** | MOV / MP4 |
| **帧率** | 30fps |
| **尺寸** | 与截图对应 |

### 视频版本
- iPhone 6.7" (1290×2796)
- iPhone 6.5" (1284×2778)  
- iPhone 5.5" (1242×2208)

### 视频内容脚本
1. 0-3秒: App启动动画，品牌展示
2. 3-8秒: 单词学习界面，卡片翻转
3. 8-15秒: 磨耳朵播放功能展示
4. 15-22秒: 学习统计界面
5. 22-30秒: 学习完成界面，行动号召
