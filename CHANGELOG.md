## [1.0.0] - 2026-02-24

### Added

  - 后台播放与锁屏控制
  - 语速调节（0.5x-1.0x）

- **数据统计**
  - 今日学习概览
  - 连续打卡天数
  - 学习趋势图表（7日/30日）

  - 词汇掌握分布

- **词库管理**
  - 3,674 个雅思词汇
  - 22 个主题章节
  - 批量导入与增量更新
  - 词库搜索与筛选

- **数据同步**
  - iCloud 同步支持
  - Core Data 本地持久化
  - CloudKit 跨设备同步

- **通知提醒**
  - 每日学习提醒
  - 学习完成通知
  - 连续打卡成就通知

- **小组件**
  - 今日进度小组件
  - 锁屏进度小组件

- **设置功能**
  - 每日学习目标设置
  - 音频播放设置
  - 学习提醒设置
  - iCloud 同步开关
  - 数据重置与导出

### Technical
- Swift 5.9 + SwiftUI 开发
- Core Data + CloudKit 数据层
- AVFoundation 音频处理
- Combine 响应式编程
- 完整单元测试覆盖

## Known Issues

### 当前版本已知问题

1. **iCloud 同步**
   - 首次同步可能较慢
   - 大量数据同步时可能出现延迟

2. **音频播放**
   - 某些设备上 TTS 语速调节不生效
   - 后台播放时偶现中断

3. **性能优化**
   - 大量单词列表滚动时有轻微卡顿
   - 统计图表渲染待优化

4. **数据导入**
   - 初次导入词库耗时较长（约 10-30 秒）

### 计划修复

- [ ] 优化 iCloud 同步性能
- [ ] 改进音频播放稳定性
- [ ] 优化列表滚动性能
- [ ] 添加导入进度指示

## Migration Guide

### 升级到 1.0.0

从测试版本升级到 1.0.0 时：

1. 建议先备份学习数据
2. 卸载旧版本后安装新版本
3. 重新导入词库
4. 学习进度会自动从 iCloud 恢复

### 数据兼容性

- 1.0.0 版本与之前的测试数据不兼容
- 建议导出旧版本数据后重新导入
