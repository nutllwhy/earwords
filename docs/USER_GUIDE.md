## 快速开始

### 首次使用指南

#### 1. 创建账户（可选）
EarWords 支持匿名使用，但创建账户可以：
- 同步学习数据到云端
- 在多台设备间同步进度
- 参与学习排行榜

**操作步骤：**
1. 打开应用，点击底部"我的"
2. 选择"账户管理"
3. 选择注册方式（Apple ID / 微信 / 邮箱）
4. 完成授权即可

#### 2. 添加你的第一批单词

**方法一：手动添加**
1. 点击首页右上角"+"按钮
2. 输入英文单词（如：ephemeral）
3. 应用会自动获取：
   - 音标 /ɪˈfemərəl/
   - 释义：短暂的、瞬息万变的
   - 例句
4. 点击"添加到生词本"

**方法二：批量导入**
1. 准备 CSV 文件，格式如下：
   ```csv
   word,phonetic,definition,example
   ephemeral,ɪˈfemərəl,短暂的,Life is ephemeral.
   serendipity,ˌserənˈdɪpəti,意外发现珍奇事物的能力,Finding this restaurant was pure serendipity.
   ```
2. 进入"设置" → "数据管理" → "导入单词"
3. 选择文件，系统自动识别格式

#### 3. 开始你的第一次学习

1. 在首页查看"今日待复习"卡片
2. 点击进入学习模式
3. 看到单词后：
   - 先尝试回忆含义
   - 点击卡片翻转查看答案
   - 根据记忆程度选择：
     - 😊 完全记得 → 下次复习时间延长
     - 🤔 有点模糊 → 正常间隔复习
     - 😵 完全不记得 → 缩短复习间隔

#### 4. 开启磨耳朵模式

1. 点击底部"磨耳朵"标签
2. 选择播放模式：
   - **单词循环**：专注听单个单词
   - **列表播放**：顺序播放所有单词
   - **智能推荐**：优先播放薄弱单词
3. 设置播放速度（0.5x - 2.0x）
4. 戴上耳机，开始沉浸式学习

## 功能详解

### 🎯 核心学习模式

#### 1. 复习模式（Review Mode）

**适用场景：** 日常复习，巩固记忆

**界面说明：**
```
┌─────────────────────────────┐
│  今日待复习: 12个            │
│  预计用时: 6分钟              │
├─────────────────────────────┤
│                             │
│        ephemeral            │
│       /ɪˈfemərəl/           │
│                             │
│     [点击翻转查看释义]        │
│                             │
├─────────────────────────────┤
│  🔊 🔁 ⭐                    │
└─────────────────────────────┘
```

**评分系统：**
| 按钮 | 含义 | 效果 |
|------|------|------|
| 😊 | 完全记得 | 间隔 × 2.5，下次14天后 |
| 🤔 | 模糊记得 | 间隔 × 1.5，下次3天后 |
| 😵 | 完全不记得 | 重置为1天，明天复习 |
| ⏭️ | 跳过 | 今日不再出现，明天再复习 |

#### 2. 测试模式（Quiz Mode）

**适用场景：** 检验学习成果，考前冲刺

**题型说明：**
- **中英选择**：给出英文，选择中文释义
- **拼写测试**：听发音，拼写单词
- **语境填空**：在句子中填入正确单词

**评分规则：**
- 答对：+10分
- 连续答对：额外+5分（连击奖励）
- 答错：显示正确答案，记录错误

#### 3. 新词学习（Learn Mode）

**适用场景：** 学习全新单词

**学习流程：**
1. **预习**：显示单词、音标、释义
2. **例句学习**：在语境中理解用法
3. **发音练习**：跟读系统，AI评分
4. **记忆强化**：通过联想记忆法加深印象
5. **小测试**：确认掌握程度

### 🎧 磨耳朵系统

#### 功能说明

磨耳朵是 EarWords 的特色功能，利用碎片时间进行听力训练。

**播放控制：**
| 手势 | 操作 |
|------|------|
| 单击 | 播放/暂停 |
| 双击左耳 | 上一个单词 |
| 双击右耳 | 下一个单词 |
| 长按 | 重复当前单词 |

**智能推荐算法：**
- 优先播放最近添加的单词
- 增加易错单词的播放频率
- 根据学习时间调整播放节奏

**场景模式：**
1. **通勤模式**：正常语速，单词+释义
2. **睡眠模式**：慢速，仅播放单词（后台播放）
3. **专注模式**：仅播放单词发音，无释义

### 📊 数据看板

#### 学习统计

**今日概览：**
- 复习单词数
- 新学单词数
- 学习时长
- 掌握程度分布

**长期趋势：**
- 词汇量增长曲线
- 记忆留存率
- 每日学习时间
- 连续打卡天数

**能力评估：**
- 词汇量估算（基于已掌握单词）
- 各等级单词掌握比例
- 易错词词频分析

## 高级功能

### 1. 自定义学习计划

**设置路径：** 我的 → 学习计划

**可配置项：**
- **每日新词量**：5-50个（推荐20个）
- **每日复习上限**：50-500个
- **学习提醒时间**：自定义提醒通知
- **休息日设置**：每周可设置1-2天休息

### 2. 单词标签系统

**用途：** 为单词添加自定义分类

**示例标签：**
- 考试类：CET-4, CET-6, 雅思, 托福, GRE
- 场景类：商务, 旅游, 学术, 日常
- 状态类：易错, 重点, 已掌握
- 自定义：任意文本标签

**使用方法：**
1. 在单词详情页点击"添加标签"
2. 输入或选择已有标签
3. 可按标签筛选复习

### 3. 导入/导出功能

**支持的格式：**
- CSV（通用格式，兼容大多数软件）
- Anki（可直接导入Anki卡组）
- JSON（完整数据备份）

**使用场景：**
- 换手机时迁移数据
- 与朋友分享单词本
- 备份重要学习数据

### 4. 云端同步

**同步内容：**
- 单词数据
- 学习进度
- 设置偏好
- 学习统计

**冲突解决：**
当多设备数据冲突时，系统会：
1. 保留最新的学习进度
2. 合并单词本（去重）
3. 优先保留较大的学习天数

### 5. 小组件（Widget）

**支持的小组件：**
- **今日复习**：快速查看待复习数量
- **随机单词**：桌面展示一个单词
- **学习进度**：今日目标完成度
- **打卡状态**：快速打卡入口

**设置方法：**
1. 长按主屏幕空白处
2. 点击"+"添加小组件
3. 选择"EarWords"
4. 选择喜欢的样式

## 数据管理

### 备份与恢复

#### 自动备份
- 开启云端同步后，数据自动备份
- 每7天自动创建本地备份

#### 手动备份
1. 进入"设置" → "数据管理"
2. 点击"创建备份"
3. 选择保存位置
4. 备份文件格式：`earwords_backup_YYYYMMDD.json`

#### 恢复数据
1. 进入"设置" → "数据管理"
2. 点击"从备份恢复"
3. 选择备份文件
4. 确认恢复（会覆盖当前数据）

### 数据导出

#### 导出为 CSV
```
word,phonetic,definition,example,tags,created_date
ephemeral,ɪˈfemərəl,短暂的,Life is ephemeral.,雅思,2024-01-15
```

#### 导出为 Anki
可直接导入 Anki 使用，保留学习进度

### 清理数据

**清理选项：**
- 清空已掌握单词（保留记录）
- 删除所有学习数据（重置账户）
- 清理缓存文件

## 故障排除

### 常见问题解决

#### 音频无法播放

**可能原因及解决：**
1. **静音模式**
   - 检查手机是否处于静音模式
   - 应用设置中开启"忽略静音模式"

2. **音频下载失败**
   - 检查网络连接
   - 进入设置重新下载音频资源

3. **蓝牙设备问题**
   - 断开并重新连接蓝牙
   - 检查蓝牙耳机电量

#### 同步失败

**排查步骤：**
1. 检查网络连接
2. 确认已登录账户
3. 尝试手动同步（下拉刷新）
4. 退出并重新登录

#### 数据丢失

**恢复方法：**
1. 检查是否有云端备份
2. 查找本地自动备份
3. 联系客服：support@earwords.app

#### 应用卡顿

**优化建议：**
1. 清理缓存：设置 → 清理缓存
2. 减少同时加载的单词数
3. 关闭不必要的动画效果
4. 重启应用

### 性能优化设置

**低性能设备建议：**
- 关闭"自动发音"
- 减少每日复习上限
- 关闭"详细统计图表"
- 使用"精简模式"

## 常见问题

### Q: EarWords 是免费的吗？
**A:** 基础功能完全免费，包括：
- 单词添加和管理
- 复习系统
- 基础统计

高级功能需要订阅：
- 云端同步
- 无限单词量
- 高级统计报告
- 发音评分

### Q: 可以离线使用吗？
**A:** 可以！大部分功能支持离线使用：
- ✅ 复习单词
- ✅ 添加新词
- ✅ 查看统计数据
- ❌ 云端同步（需联网）
- ❌ 发音下载（需提前下载）

### Q: 如何删除单词？
**A:** 
1. 在单词详情页点击右上角"..."
2. 选择"删除单词"
3. 确认删除（可撤销，24小时内）

### Q: 支持哪些词典？
**A:** 
- 内置：简明英汉词典
- 在线：有道词典、牛津词典（需联网）
- 可自定义导入词典

### Q: 如何修改学习提醒时间？
**A:** 
1. 进入"设置" → "通知设置"
2. 点击"学习提醒"
3. 选择新的提醒时间
4. 确认系统通知权限已开启

### Q: 单词上限是多少？
**A:** 
- 免费用户：500个单词
- 订阅用户：无限制
- 已达上限时，可以：
  - 删除不常用的单词
  - 升级到订阅版
  - 导出单词后清空部分数据
