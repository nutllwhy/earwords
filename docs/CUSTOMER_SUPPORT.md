| 类型 | 描述 | 处理时效 |

| 账号问题 | 登录、同步、数据丢失 | 4小时内回复 |

```

我是客服小E，已经查看了您的问题，现在开始为您处理。
```

### 常见问题回复

#### 1. 同步问题

**用户：** "我的数据同步不了"

**客服回复：**
```
感谢您的反馈！数据同步问题通常由以下几种情况导致：

1️⃣ 网络连接问题
   → 请检查WiFi或移动数据是否正常

2️⃣ 账号未登录
   → 请确认是否已登录账户（我的 → 账户管理）

3️⃣ 同步冲突
   → 尝试下拉首页手动刷新

请先尝试以上方法，如果问题仍然存在，请告诉我：
- 您的设备型号
- App版本号（设置 → 关于 中查看）
- 具体的错误提示

我会立即为您排查！
```

#### 2. 退款申请

**用户：** "我想申请退款"

**客服回复：**
```
理解您的需求，我来帮您处理退款事宜。

首先，请确认您的购买渠道：

📱 iOS用户（App Store购买）：
   退款需要向Apple申请，我们协助您操作：
   1. 访问 reportaproblem.apple.com
   2. 登录您的Apple ID
   3. 找到EarWords购买记录
   4. 点击"报告问题" → 选择"请求退款"
   5. 说明退款原因
   
   或直接拨打 Apple 客服：400-666-8800

💳 其他渠道购买：
   请提供您的订单号，我会立即为您处理。

需要我发送详细的退款教程到您的邮箱吗？
```

#### 3. 数据丢失

**用户：** "我的单词数据不见了！"

**客服回复：**
```
非常理解您的着急，数据丢失我们会全力帮您恢复！

请先不要操作App，按以下步骤检查：

🔍 第一步：确认登录状态
   - 检查是否登录了正确的账户
   - 尝试退出后重新登录

🔍 第二步：检查同步状态
   - 下拉首页刷新
   - 查看最后一次同步时间

🔍 第三步：检查云端备份
   - 进入 设置 → 数据管理 → 恢复数据
   - 查看是否有历史备份

如果以上方法无效，请提供：
- 用户ID（设置 → 关于 → 用户ID）
- 大概丢失了多少单词
- 最后一次看到数据的时间

我们的技术团队会立即从服务器端帮您恢复！
```

#### 4. 功能咨询

**用户：** "这个XX功能怎么用？"

**客服回复：**
```
感谢您的咨询！关于【功能名称】的使用方法：

📖 快速操作指南：
1. 进入【页面路径】
2. 点击【按钮/区域】
3. 选择【选项】
4. 完成【操作】

🎥 视频教程：
我已发送详细教程到您的邮箱，包含图文步骤和视频演示。

💡 小技巧：
- 技巧1：xxx
- 技巧2：xxx

还有其他问题随时问我哦！
```

#### 5. 投诉处理

**用户：** "你们这App太难用了！"

**客服回复：**
```
非常抱歉给您带来了不好的体验，我真诚地向您道歉。

为了更好地帮助您解决问题，能否告诉我具体是哪里让您不满意？

是：
□ 操作复杂，不知道怎么用
□ 遇到了Bug或闪退
□ 功能不符合预期
□ 其他原因

您的反馈对我们非常重要，我会：
1. 立即帮您解决当前问题
2. 将您的建议转达给产品团队
3. 如有需要，为您申请补偿

请告诉我具体情况，好吗？
```
