## 视频录制规范

### 技术标准

| 项目 | 规格 |
|------|------|
| 分辨率 | 1920×1080 (1080p) 或更高 |
| 帧率 | 30fps |
| 格式 | MP4 (H.264编码) |
| 音频 | 48kHz, 立体声 |
| 时长控制 | ±5秒 |
| 文件命名 | `EW_类型_版本_日期.mp4` |

### 视觉风格

**色彩方案：**
- 主色：#4A90E2（科技蓝）
- 辅色：#50E3C2（活力绿）
- 背景：#F5F7FA（浅灰）
- 文字：#2C3E50（深灰）

**字体规范：**
- 标题：PingFang SC Bold / 36pt
- 正文：PingFang SC Regular / 18pt
- 字幕：PingFang SC Regular / 24pt

**动效要求：**
- 转场：淡入淡出 0.3s
- 元素动画：弹性缓出
- 手势提示：高亮+放大效果

### 音频规范

**配音要求：**
- 语速：每分钟约150字
- 语调：亲切、清晰、有节奏感
- 背景音：轻音乐，音量-20dB

**音效使用：**
- 点击：清脆提示音
- 成功：正向反馈音
- 转场：轻微过渡音

### 推荐工具

**录制工具：**
| 工具 | 平台 | 特点 |
|------|------|------|
| ScreenFlow | macOS | 专业录屏+剪辑一体 |
| Camtasia | Win/Mac | 功能全面，模板丰富 |
| OBS Studio | 全平台 | 免费开源，高度可定制 |
| iOS自带录屏 | iOS | 轻量便捷 |

**后期工具：**
- 剪辑：Final Cut Pro / Premiere Pro / DaVinci Resolve
- 动效：After Effects / Principle
- 字幕：ArcTime / 剪映

## 通用后期规范

### 字幕规范

**格式：**
- SRT或内嵌字幕
- 每行不超过12个字
- 停留时间不少于2秒
- 转场时提前0.2秒消失

**样式：**
- 字体：PingFang SC / Noto Sans
- 大小：24-28pt
- 颜色：白色
- 描边：黑色1px
- 位置：底部居中，距底边10%

### 导出设置

**iOS/Android预览版：**
- 分辨率：1080×1920（竖屏9:16）
- 码率：8Mbps
- 文件大小：控制在50MB以内

**网页/宣传版：**
- 分辨率：1920×1080（横屏16:9）
- 码率：10Mbps
- 文件大小：控制在100MB以内

### 多语言版本

**英文版调整：**
- 语速加快至180词/分钟
- 字幕行数增加
- 文化场景适配

**其他语言：**
- 根据语言长度调整画面停留时间
- RTL语言需镜像界面
