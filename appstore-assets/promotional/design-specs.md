# 推广素材设计规范

## 应用图标 (App Icon)

### 规格要求
| 规格 | 尺寸 | 用途 |
|------|------|------|
| App Store | 1024×1024 px | 应用商店展示 |
| iPhone | 180×180 px | @3x 主屏幕 |
| iPhone | 120×120 px | @2x 主屏幕 |
| iPad | 167×167 px | @2x 主屏幕 |
| Spotlight | 80×80 px | 搜索建议 |
| Settings | 58×58 px | 设置页面 |

### 设计概念

**核心理念**: 耳朵 + 书本 + 声音波

**设计元素**:
- 背景: 深蓝渐变 (#1A1A2E → #16213E)
- 主图标: 抽象化的耳朵轮廓 + 书本/卡片形状
- 强调色: 青绿色 (#4ECCA3) 表示声音/学习
- 圆角: 与iOS图标规范一致 (22%圆角)

**字体**: 无文字，纯图形标识

## 社交媒体推广图

### Instagram / 小红书 (1080×1080)
**内容**: 功能亮点展示
- 主标题: 大字标题 + 简洁副标题
- 设备展示: 手机mockup展示App界面
- 背景: 品牌渐变或纯色
- 品牌Logo: 底部或角落

### Twitter / 微博 / 即刻 (1200×675)
**内容**: 新闻公告或更新
- 横向布局，适合信息流
- 简洁有力的标题
- 视觉冲击力强的配色

### 微信文章封面 (900×383)
**内容**: 公众号文章头图
- 居中构图
- 标题醒目
- 品牌色调统一

## 设计模板规范

### 色彩系统
```
主色:
- Deep Navy: #1A1A2E
- Dark Blue: #16213E
- Medium Blue: #0F3460

强调色:
- Teal: #4ECCA3
- Light Teal: #7ED6C1

辅助色:
- Orange: #FF6B6B (警告/强调)
- Yellow: #FFE66D (成就/徽章)
- Purple: #9B59B6 (特殊功能)

中性色:
- White: #FFFFFF
- Light Gray: #E0E0E0
- Dark Gray: #333333
```

### 字体规范
- **标题**: San Francisco Display / PingFang SC Semibold
- **正文**: San Francisco Text / PingFang SC Regular
- **强调**: San Francisco Semibold / PingFang SC Medium

### 间距系统
- 小间距: 8px
- 中间距: 16px
- 大间距: 24px
- 超大间距: 48px

## 导出规范

### 图片格式
- **Web/社交**: PNG (透明背景) 或 JPG (照片)
- **打印**: PDF 或 PNG (300dpi)
- **动画**: GIF 或 MP4

### 命名规范
```
icon-1024x1024.png
social-instagram-01.png
social-twitter-01.png
poster-feature-algorithm.png
poster-feature-vocabulary.png
...
```
