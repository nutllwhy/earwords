## 二、收入预测模型

### 2.1 订阅收入计算

**ARPU（每用户平均收入）计算**：

```
年度 ARPU = (月度价格 × 月度占比 × 平均月数 × 12) 
          + (年度价格 × 年度占比)
          + (终身价格 × 终身占比 / 3)  # 终身按 3 年摊销

= (18 × 0.15 × 3) + (68 × 0.60) + (128 × 0.25 / 3)
= 8.1 + 40.8 + 10.67
= ¥59.57/年/付费用户
```

| 36 个月 | ¥2,507,000 | ¥523,000 | ¥1,984,000 | 79% | +¥25,000,000 |

### 4.2 盈亏平衡点

**计算**：
```
盈亏平衡点 = 固定成本 / (1 - 变动成本率)

假设：
- 固定成本：¥3,000/月
- 变动成本率：约 20%（不含苹果抽成则为 15%）

盈亏平衡收入 = 3,000 / (1 - 0.20) = ¥3,750/月
```

**结论**：
- 预计在 **第 2 个月** 达到月度盈亏平衡
- 预计在 **第 6 个月** 收回前期投入（¥10 万）

## 八、财务规划建议

### 8.1 短期（0-6 个月）

1. **严格控制成本**：保持 lean 运营，避免过早扩张
2. **数据驱动决策**：每日监控付费转化漏斗，快速迭代
3. **储备现金流**：保持至少 6 个月运营资金

### 8.2 中期（6-18 个月）

1. **规模化推广**：验证获客模型后，增加推广投入
2. **优化利润率**：与苹果协商降低抽成，探索网页支付
3. **团队扩张**：招聘运营、客服，释放创始人时间

### 8.3 长期（18 个月+）

1. **多元化收入**：拓展 B 端市场（学校/机构合作）
2. **国际化**：进入东南亚等英语教育市场
3. **资本化**：视情况启动融资或寻求并购机会

## 附录：财务模型 Excel 公式

```excel
# 订阅收入
=MAU * 付费率 * (月度价*月度占比*3 + 年度价*年度占比 + 终身价*终身占比/3) / 12

# 广告收入
=DAU * 免费用户占比 * 人均展示 * eCPM / 1000 * 30

# 总成本
=固定成本 + 推广费用 + 服务器成本 + 苹果抽成

# 净利润
=总收入 - 总成本 - 税费

# LTV
=ARPU * 平均订阅时长 / 流失率

# CAC
=推广费用 / 新增付费用户
```
