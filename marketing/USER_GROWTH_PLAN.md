### 二、推送通知策略（不打扰但有效）

#### 推送原则

```
1. 价值优先：每条推送都提供明确价值
2. 个性化：基于用户行为和数据定制
3. 时机恰当：在用户可能接受的时间发送
4. 频率控制：每日最多2条，避免轰炸
5. 可关闭：尊重用户选择，提供关闭选项
```

#### 推送类型与场景

| 推送类型 | 触发条件 | 发送时间 | 文案示例 |
|----------|----------|----------|----------|

| 流失唤醒 | 3天未打开 | 19:00 | "想念你的第3天，送你7天会员快回来吧" |

#### 优先级1：雅思/托福备考博主

```

```
画像：
- 粉丝量：5万-100万
- 内容：英语学习方法、资源推荐
- 平台：B站、抖音、小红书
- 受众：英语学习爱好者

合作形式：
- 学习方法分享

- 粉丝互动：评论区活跃度高

### 三、推广激励机制

#### KOL分层体系

```
【种子KOL】粉丝1万-5万
- 合作模式：产品置换为主，少量付费
- 合作目标：内容积累、口碑传播
- 权益：年费会员、新功能内测、周边礼品

【成长KOL】粉丝5万-20万
- 合作模式：付费合作+效果奖励
- 合作目标：精准获客、转化提升
- 权益：基础费用+CPA奖励（按下载量）

【头部KOL】粉丝20万+
- 合作模式：定制化深度合作
- 合作目标：品牌曝光、大规模获客
- 权益：项目制费用+长期合作框架
```

#### 激励方案

```
【基础费用】
根据粉丝量和互动率定价：
- 1-5万粉：500-2000元
- 5-10万粉：2000-5000元
- 10-30万粉：5000-15000元
- 30万+粉：15000-50000元

【效果奖励】
- 专属优惠码带来的下载量 × 5元/个
- 带来的付费转化 × 20%佣金
- 单条内容曝光破10万：额外奖励1000元

【长期激励】
- 季度合作3次以上：费用上浮20%
- 年度合作10次以上：升级为品牌大使
- 品牌大使专属：年费保底+独家权益
```

#### 合作流程

```
1. 筛选与触达（1-3天）
   → 根据画像筛选KOL名单
   → 发送合作邀约

2. 沟通与确认（3-7天）
   → 介绍产品与方案
   → 确认合作形式与费用
   → 签订合同

3. 产品体验（7-14天）
   → 提供产品会员
   → 提供素材支持
   → 答疑与沟通

4. 内容产出（7天）
   → 大纲审核（如有需要）
   → 内容审核（确保合规）
   → 正式发布

5. 效果追踪（30天）
   → 数据监测（曝光、互动、转化）
   → 费用结算
   → 合作复盘
```
