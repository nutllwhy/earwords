## 📱 账户与同步

### Q1: 一定要注册账户才能使用吗？
**A:** 不是必须的。EarWords 支持**匿名使用**，所有数据会保存在本地。但注册账户后可以：
- 在不同设备间同步数据
- 备份学习进度到云端
- 参与排行榜和成就系统

### Q2: 如何在多台设备间同步数据？
**A:** 
1. 在所有设备上登录同一个账户
2. 确保开启"云端同步"（设置 → 账户 → 同步设置）
3. 下拉首页即可手动同步
4. 系统也会每15分钟自动同步一次

### Q3: 同步时数据冲突怎么办？
**A:** 系统会智能合并数据：
- 学习进度：保留最新的记录
- 单词本：自动去重合并
- 设置：以最后修改的设备为准

如果仍有疑问，可以联系客服手动处理。

### Q4: 忘记了登录密码怎么办？
**A:** 
- Apple ID 登录：通过 Apple 找回
- 微信登录：重新授权即可
- 邮箱登录：在登录页点击"忘记密码"，通过邮件重置

### Q5: 可以注销账户吗？
**A:** 可以。在"设置" → "账户管理" → "注销账户"中操作。
- 注销后会删除云端所有数据
- 本地数据不受影响（可继续使用匿名模式）
- 30天内可联系客服恢复账户

## 📚 学习与复习

### Q6: SM-2算法是什么？为什么我的复习间隔这么长/短？
**A:** SM-2 是科学的间隔重复算法，根据你的记忆表现动态调整：
- **答对且记得牢**：间隔越来越长（几天→几周→几个月）
- **答错或模糊**：间隔缩短或重置

如果觉得间隔不合适，可以在"设置" → "学习计划"中调整"记忆难度系数"。

### Q7: 每天应该复习多少单词？
**A:** 建议设置：
- **每日新词**：10-20个（初学者10个，进阶者20个）
- **每日复习**：按系统推荐，通常为新词的3-5倍

可以在"设置" → "学习计划"中调整。

### Q8: 复习时"完全不记得"会有什么影响？
**A:** 
- 该单词会重置为"新词"状态
- 明天会再次复习
- 连续答错多次不会惩罚，放心按实际情况选择

### Q9: 可以暂停复习吗？比如考试周太忙了。
**A:** 可以！有两种方式：
1. **暂停模式**：设置 → 学习计划 → 开启"暂停复习"，最多暂停7天
2. **休息日**：设置每周1-2天为休息日，系统自动跳过

暂停期间，已安排的复习会顺延，不会消失。

### Q10: 为什么有些单词一直出现？
**A:** 可能原因：
1. 你总是选择"模糊记得"或"不记得"，系统判定你需要加强记忆
2. 这是新添加的单词，前3天会密集复习
3. 该单词被标记为"重点"

如果确实已掌握，可以标记为"已掌握"或调整难度。

## 🎧 磨耳朵功能

### Q11: 磨耳朵可以后台播放吗？
**A:** 可以！开启方式：
1. 进入磨耳朵页面
2. 点击设置图标
3. 开启"后台播放"
4. 即使切换到其他应用或锁屏，也会继续播放

### Q12: 如何设置播放速度？
**A:** 
- 在磨耳朵页面，点击右下角的速度按钮
- 可选：0.5x / 0.75x / 1.0x / 1.25x / 1.5x / 2.0x
- 建议：新词用0.75x，复习用1.0x，挑战用1.5x

### Q13: 磨耳朵支持离线吗？
**A:** 支持，但需要提前下载音频：
1. 在 WiFi 环境下打开磨耳朵
2. 系统会自动缓存已添加单词的音频
3. 也可在"设置" → "资源管理"中批量下载

### Q14: 可以只听不显示界面吗？
**A:** 可以，这是"专注模式"：
- 磨耳朵设置 → 开启"专注模式"
- 屏幕会变黑，只播放音频
- 适合睡前或专注场景

### Q15: 耳机控制怎么用？
**A:** 
- **单击**：播放/暂停
- **双击**：下一词（可在设置中改为上一词）
- **三击**：上一词
- **长按**：重复当前单词

## 📝 单词管理

### Q16: 如何批量导入单词？
**A:** 
1. 准备 CSV 文件，格式：
   ```
   word,definition
   apple,苹果
   banana,香蕉
   ```
2. 发送到手机（邮件、微信、AirDrop等）
3. 用 EarWords 打开文件
4. 确认导入预览，点击"确认导入"

### Q17: 导入时提示格式错误怎么办？
**A:** 检查以下几点：
- 文件编码为 UTF-8
- 使用英文逗号分隔
- 每条数据一行，不跨行
- 特殊符号用引号包裹

也可以使用模板文件：在导入页面下载"示例CSV"。

### Q18: 可以给单词添加图片吗？
**A:** 目前暂不支持自定义图片，但系统会自动为单词匹配：
- 相关配图（释义场景）
- 助记图片（记忆法配图）

在单词详情页点击"更多资源"查看。

### Q19: 如何删除误添加的单词？
**A:** 
1. 进入单词详情页
2. 点击右上角"..."菜单
3. 选择"删除单词"
4. 确认删除（24小时内可在"最近删除"中恢复）

### Q20: 单词发音可以更换吗？
**A:** 可以：
- 设置 → 发音设置 → 选择发音人
- 可选：英式男声 / 英式女声 / 美式男声 / 美式女声
- 部分高级发音需要订阅

## 🔧 技术与性能

### Q26: 应用占用存储空间太大怎么办？
**A:** 
1. 设置 → 清理缓存
2. 资源管理 → 删除不常用的发音包
3. 设置 → 减少"历史记录保存天数"
4. 卸载重装（记得先同步数据）

### Q27: 耗电快怎么优化？
**A:** 
- 关闭"后台持续同步"
- 减少动画效果（设置 → 显示设置）
- 磨耳朵时开启"省电模式"
- 不要长时间停留在学习界面（会自动保持亮屏）

### Q28: 支持 iPad/Mac 吗？

## 🎯 学习方法

### Q31: 零基础适合用 EarWords 吗？
**A:** 适合！建议：
1. 从"基础词汇包"开始（内置500词）
2. 每日新词设置为5-10个
3. 开启"详细释义"和"例句"
4. 多使用磨耳朵功能培养语感

### Q32: 备考雅思/GRE适合吗？
**A:** 非常适合！特色功能：
- 内置各考试核心词库
- 词根词缀记忆法
- 同义词/反义词关联
- 高频考点标记

### Q33: 如何使用最高效？
**A:** 推荐学习流程：
1. **早晨**：复习今日待复习单词（10分钟）
2. **碎片时间**：磨耳朵模式（通勤、排队时）
3. **晚上**：学习新单词（15分钟）
4. **睡前**：快速过一遍今日学习

### Q34: 单词记不住怎么办？
**A:** 试试这些方法：
- 开启"助记法"提示
- 添加个人笔记
- 自己造句
- 多用磨耳朵听发音
- 降低每日新词量，加强复习

### Q35: 打卡断了怎么办？
**A:** 别担心！
- 连续打卡奖励只是鼓励，不影响实际学习
- 可以使用"补签卡"（通过分享获得）
- 重点在于持续学习，而非连续天数
