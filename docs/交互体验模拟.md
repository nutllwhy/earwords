【交互细节说明】

1. 卡片翻转
   - 点击"显示释义"：卡片向上滑动展开释义
   - 点击"显示例句"：释义下方继续展开例句
   - 展开时有 0.3s 的平滑动画

2. 评分反馈
   - 点击数字后，卡片向右侧滑出
   - 新卡片从左侧滑入
   - 底部进度条实时更新

3. 手势操作（高级功能）
   - 左滑：跳过当前单词
   - 右滑：标记重点（加入收藏）
   - 上滑：快速显示释义
   - 下滑：返回上一个单词

4. 磨耳朵后台播放
   - 锁屏显示当前单词
   - 耳机控制：单击暂停/播放，双击下一首
   - 控制中心显示播放进度
